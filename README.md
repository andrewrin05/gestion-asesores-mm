# Gestión y Asesores M&M SL

## Descripción del Proyecto

Gestión y Asesores M&M SL es una empresa dedicada a ofrecer servicios de seguros personalizados. Este proyecto es un sitio web informativo que proporciona a los usuarios una visión general de los servicios ofrecidos, así como información sobre la empresa y cómo contactarla.

## Estructura del Proyecto

El proyecto está organizado de la siguiente manera:

- **src/**: Contiene todos los archivos fuente del sitio web.
  - **index.html**: Punto de entrada principal del sitio.
  - **pages/**: Contiene las diferentes páginas del sitio.
    - **home.html**: Página de inicio con una visión general de los servicios.
    - **about.html**: Información sobre la empresa, su misión y equipo.
    - **services.html**: Lista de servicios de seguros ofrecidos.
    - **contact.html**: Formulario de contacto y detalles de contacto.
  - **components/**: Componentes reutilizables del sitio.
    - **header.html**: Estructura del encabezado del sitio.
    - **footer.html**: Estructura del pie de página del sitio.
  - **styles/**: Archivos de estilo CSS.
    - **main.css**: Estilos principales del sitio.
  - **scripts/**: Archivos de JavaScript.
    - **main.js**: Código JavaScript para características interactivas.
- **data/**: Contiene datos estructurados.
  - **services.json**: Datos sobre los servicios ofrecidos.
- **docs/**: Documentación adicional.
  - **privacy.md**: Política de privacidad de la empresa.
- **package.json**: Archivo de configuración para npm.
- **.gitignore**: Archivos y directorios a ignorar por el control de versiones.

## Instalación

1. Clona el repositorio en tu máquina local.
2. Navega al directorio del proyecto.
3. Ejecuta `npm install` para instalar las dependencias necesarias.

## Uso

Abre `src/index.html` en tu navegador para ver el sitio web en acción. Navega a través de las diferentes páginas para explorar los servicios y obtener más información sobre Gestión y Asesores M&M SL.
